## Features

-  Job discovery with smart filters  
-  Personalized job recommendations  
-  Save and track applications  
-  Easy apply with resume autofill  
-  Built with Next.js App Router and optimized fonts via [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) using [Geist](https://vercel.com/font)
